Tutorial sobre ambiente de desenvolvimento Xampp

A tecnologia Xampp é um ambiente de desenvolvimento que contém pacotes de software que são utlizados, para fazer sistemas web, ele possui suporte as linguagens PHP e PERL, possui mysql e servidores Apache e Tomcat .

É uma linguagem de script multi-paradigma, baseada em protótipo que é dinâmica, e suporta estilos de programação orientado a objetos, imperativo e funcional

A Ipinfo é uma api de geolocalização que permite usuário saber as coordenados do endereço ip do laptop , computador e servidores e aparelho celulares

O PHP é uma linguagem poderosa e um interpretador, seja incluído em um servidor web como um módulo ou executado separadamente como binário CGI, é possível acessar arquivos, executar comandos e abrir conexões de rede no servidor.

Construir o Site de notícias com geolocalização

1º Fase

Acessar site e Baixar o Xampp

Copie o link para baixar Xampp para o Windows 7:(https://www.apachefriends.org/pt_br/download_success.html) Depois que baixar o arquivo dá um duplo clique no executável do Xampp. E depois é só clicar Next em todos fases. Dica leia a documentação dos Xampp é recomendável

2ºFase

Acessar site e Baixar o Codeigniter

Copie o link para baixar o Framework Codeigniter para Windows7:(https://www.codeigniter.com). Assim,que terminar descompactar com o sistema 7-zip segue o link caso você não tenha instalado em seu Computador (http://www.7-zip.org/)

3ºFase

Colocar o Framework Codeigniter na pasta htdocs do Xampp

Esta fase consiste em colocar o Framework Codeigniter na pasta htdocs e abrir com editor notepad++ ou sublime para fazer o sistema em php
